# Developer Blog 
This project for developer blog.
It is render mdx file to blog page.
So, you can write blog with markdown syntax.

You can deploy this project anywhere you want. (Vercel, Amplify, etc...)


## Tech Stack
- TypeScript
- React.js
- Next.js
- Tailwind CSS
